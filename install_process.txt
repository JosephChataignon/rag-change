Install python modules

create admin with `python manage.py createsuperuser`
    currently username: joseph, password: hjgfpole

apply migrations with `python manage.py migrate`

Go to ragchange/config/local.yaml to set local config. 
Settings in local.yaml override defaults.yaml

Run tests with `python3 manage.py test`

Run server with `python3 manage.py runserver` (testing)

For production, set DJANGO_DEBUG=False and configure allowed hosts 
in the environment variable DJANGO_ALLOWED_HOSTS (comma separated).

Collect static files with `python manage.py collectstatic`
In your server config, add a location block to serve static files. For example, in nginx:
    location /static/ { # This parameter should match STATIC_URL
        alias /home/change/rag-change/staticfiles/; # This parameter should match STATIC_ROOT
    }
